# Object_oriented_programming-language-and-lab-cse_course_111-112-
Welcome to the repository for CSE 111 & 112: Object-Oriented Programming Language and Lab!This repository is a structured collection of my coursework, assignments, and implementations developed entirely in C++. 

It reflects my progression through core Object-Oriented Programming (OOP) concepts, covering both theoretical understanding and practical application.From foundational topics explored in lectures to hands-on lab exercises, this repository documents my learning journey and demonstrates how key OOP principles are applied in real code.

📂 Repository Structure

The repository is organized into distinct directories to easily navigate between theoretical concepts and hands-on coding tasks:

    /THEORY CLASS: Contains code examples,snippets, and concepts covered during lecture sessions. This focuses on understanding the core pillars of OOP: Encapsulation, Inheritance, Polymorphism, and Abstraction.
    /LAB: Includes practical lab assignments, hands-on coding exercises, and problem-solving tasks implemented in C++.
    /Circle Class: A dedicated module implementing a Circle class, demonstrating object instantiation, constructors, member functions, and access control.
